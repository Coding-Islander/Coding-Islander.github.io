# Workshop Landing Page Generation Guidelines

## 🎯 Objective
Generate high-converting landing pages for technical workshops.
The content MUST position the offering as a hands-on workshop and NOT as a formal course or training program.

---

## 🚫 Strict Rules (MANDATORY)

DO NOT use the following terms:
- course
- training program
- certification
- diploma
- exam
- assessment

USE instead:
- workshop
- session
- hands-on
- live practice
- interactive

The page must feel like:
- a practical experience
- a live guided session
- a real-world skill-building event

---

## ✍️ Writing Style

- Clear and conversational
- Focus on real-world value
- Avoid academic tone
- Use short paragraphs and bullet points
- Speak directly to developers
- Show value to developers

---

## 💡 Positioning Guidelines

The page should:
- Feel practical, not theoretical
- Emphasize doing over learning
- Focus on solving real developer problems
- Build confidence and clarity
